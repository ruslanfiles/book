# TrendMasterEA — скальпинг Expert Advisor для XAUUSD M5 (XM, MT5)

Готовый к компиляции и прогону в Strategy Tester скальпинг-советник по золоту.
Язык: **MQL5** (для MT5). Файл: `TrendMasterEA.mq5`.
Настроен под **XAUUSD, таймфрейм M5, скальпинг**.

> 🔴 **ОСОБОЕ ПРЕДУПРЕЖДЕНИЕ ПРО БЮДЖЕТ $15.**
> $15 — экстремально малый депозит для золота. Минимальный лот XAUUSD у XM = 0.01
> (это 1 унция), и один пункт цены (0.01) = $0.01 P&L. Реалистичный скальп-стоп
> на M5 (например 100-150 пунктов = $1.00-$1.50) при 0.01 лота = 7-10% депозита
> за одну сделку. Две-три подряд убыточные — и депозит близок к нулю/margin call.
> Встроена защита `InpMinEquityStop` (по умолчанию $15): при падении эквити ниже
> порога бот полностью останавливается. **Используйте только как демо/учебный
> или заведомо рисковый капитал, потерю которого готовы принять.**

## 🟢 ДВЕ ВЕРСИИ СОВЕТНИКА

В репозитории два файла:

1. **`TrendMasterEA.mq5`** — скальпинг, XAUUSD, **M5** (описание ниже).
2. **`TrendMasterEA_LongTerm.mq5`** — **долгосрочная/позиционная** торговля, **H4 + D1/W1**.

### Долгосрочная версия (TrendMasterEA_LongTerm.mq5)
Тот же движок и модули, но настроен под большой таймфрейм:

| Область | Значение |
|---|---|
| Рабочий ТФ | **H4** |
| Тренд-фильтры | **D1** (`InpUseMTFFilter=true`) + **W1** (`InpUseTrendTF2=true`) |
| EMA-биас | **21 / 50** на H4 |
| Триггер | **откат по Stochastic %K/%D** в сторону тренда D1 (частые входы, свинг) |
| ADX | **вкл**, порог 20 |
| SL / TP | **2.5× / 6.0× ATR** (крупный R:R ~2.4, «дать прибыли расти») |
| Трейлинг | 3.0× ATR — тянем тренд |
| Сессия/выходные | фильтры выключены (позиции держим долго) |
| Magic Number | 20260706 (иной, чтобы не путать со скальпером) |

> ### 🔴 КРИТИЧНО ПРО $15 И БОЛЬШОЙ ТАЙМФРЕЙМ
> Большой ТФ = **широкие стопы**. На H4 у золота ATR легко $15–40, а стоп
> `2.5×ATR` = **$40–100 движения цены**. Для 0.01 лота (мин. лот) это **$40–100
> потенциального убытка на одну сделку** — а депозит всего $15. Это значит:
> **стоп-лосс по деньгам БОЛЬШЕ всего счёта → маржин-колл почти гарантирован.**
>
> Долгосрочная торговля золотом на $15 **физически невозможна** при минимальном
> лоте. Реальные варианты:
> - завести счёт **крупнее** (для 0.01 лота свинг-золота ориентир — от $1000–2000);
> - или запустить этот же EA на **менее дорогом/волатильном инструменте** (например
>   форекс-пара EURUSD H4 — там 0.01 лота со стопом обходится в разы дешевле);
> - на $15 долгосрочку имеет смысл тестировать **только на демо**.

> ⚠️ **Дисклеймер.** Это инструмент автоматизации, а не финансовая рекомендация.
> Торговля на форекс/CFD сопряжена с риском потери капитала. Любую стратегию нужно
> самостоятельно протестировать и проверить перед использованием реальных денег.
> Значения параметров ниже — стартовая точка для тестирования, а не готовая
> прибыльная настройка.

## 1. Что реализовано

Конфигурация по умолчанию (всё меняется через input):

| Область | Реализация |
|---|---|
| **Тип стратегии** | Скальпинг по тренду (fast trend scalping) |
| **Инструмент** | **XAUUSD (золото)** — запускать на графике XAUUSD |
| **Рабочий ТФ** | **M5** (старший для MTF-фильтра — M15) |
| **Вход** | **Тренд-биас по EMA(8)/EMA(21)** + **частый триггер по пересечению Stochastic %K/%D** в сторону тренда. RSI — опц. фильтр. ADX/MACD/Bollinger — опц. (по умолч. выкл.) |
| **MTF-фильтр** | По умолчанию **выкл** (только анализ на панели), можно включить |
| **Выход** | ATR-based SL/TP (1.5×/**2.0×**, асимметр. R:R), break-even, трейлинг, частичное закрытие, реверс |
| **Риск** | Фикс. лот 0.01 (для $15), дневной лимит, макс. просадка, стоп по мин. эквити $15 |
| **Фильтры** | Спред (≤40 пунктов), сессия, ATR-волатильность, новости USD/EUR, выходные |
| **Мониторинг** | Лог в файл + журнал, Telegram, Push, панель на графике |

> **Про «пункты» на золоте.** У XAUUSD `Digits = 2`, поэтому 1 «пункт» (pip) в
> параметрах фильтров = 1 пункт цены = **0.01 USD**. Функция `PipSize()` это
> учитывает автоматически. Т.е. `InpMaxSpreadPips = 40` означает спред ≤ $0.40,
> `InpAtrMinPips = 20` означает ATR ≥ $0.20 и т.д.

Модульная структура кода (функции сгруппированы по модулям 1–8): сигналы, фильтры,
риск, исполнение, управление позициями, нормализация, логирование/уведомления, панель.

### СКАЛЬПИНГ-ИНДИКАТОРЫ (доп. подтверждение входа)
Добавлены популярные для M5-скальпинга индикаторы (по материалам открытых источников
о скальпинге золота/форекс). Все включаются/выключаются отдельно.

| Параметр | По умолч. | Назначение |
|---|---|---|
| `InpUseStochastic` | true | **Стохастик** — импульс + отсечение перекупленности/перепроданности. BUY: %K ниже 80 и %K ≥ %D; SELL: %K выше 20 и %K ≤ %D |
| `InpStochK` / `InpStochD` / `InpStochSlowing` | 5 / 3 / 3 | Параметры Стохастика (быстрые, под M5) |
| `InpStochUpper` / `InpStochLower` | 80 / 20 | Уровни перекупл./перепрод. |
| `InpUseMACD` | true | **MACD** — подтверждение направления импульса (BUY: main>signal; SELL: main<signal) |
| `InpMacdFast` / `InpMacdSlow` / `InpMacdSignal` | 12 / 26 / 9 | Параметры MACD |
| `InpUseBollinger` | false | **Bollinger Bands** — не входить на растянутом крае канала (анти-погоня за ценой) |
| `InpBBPeriod` / `InpBBDeviation` | 20 / 2.0 | Параметры Bollinger |

> **Новая логика входа (после проблемы «год без сделок»):**
> раньше триггером было редкое пересечение EMA, на которое навешивались обязательные
> ADX+Stochastic+MACD — совпадение всего в один бар почти невозможно, отсюда 0 сделок.
> Теперь:
> 1. **Тренд-биас** — по состоянию EMA(8) vs EMA(21) (не по редкому кроссу);
> 2. **Триггер** — пересечение Stochastic %K/%D в сторону тренда (частое событие);
> 3. **Опциональные фильтры** — RSI (вкл), ADX/MACD/Bollinger (по умолчанию **выкл**,
>    чтобы не блокировать; включайте по одному для повышения качества).
>
> Если `InpUseStochastic = false`, триггером снова станет пересечение EMA.
> Каждый отказ фильтра пишется в лог с причиной и цифрами.

## 3. Установка на демо-счёт (XM MT5)

1. Скопируйте `TrendMasterEA.mq5` в папку данных терминала:
   **Файл → Открыть каталог данных → `MQL5/Experts/`**.
2. Откройте **MetaEditor** (F4), выберите файл, нажмите **Compile** (F7).
   Ошибок компиляции быть не должно (0 errors).
3. Вернитесь в терминал MT5, обновите список советников (правый клик → Обновить).
4. Откройте график **XAUUSD** на таймфрейме **M5**.
5. Перетащите советник на график. На вкладке **Общие** включите
   «Разрешить алготрейдинг». Кнопка **АлгоТрейдинг** вверху должна гореть.
6. Для Telegram: **Сервис → Настройки → Советники →** отметьте
   «Разрешить WebRequest для URL» и добавьте `https://api.telegram.org`.
7. Настройте input-параметры и нажмите ОК.
8. Убедитесь, что у брокера символ называется именно `XAUUSD` (у XM обычно так;
   бывают суффиксы вроде `XAUUSD.` или `GOLD` — тогда запускайте на этом символе).

## 4. Тестирование в Strategy Tester

- **Режим модели:** для скальпинга на M5 обязательно **«Every tick based on real
  ticks»**. Модель «1 minute OHLC» даст завышенно-оптимистичный результат, т.к. не
  учитывает движение цены и спред внутри бара — для скальпинга это критично.
- **Период:** 3–6 месяцев достаточно (на M5 сделок много), но проверьте разные
  фазы рынка золота (тренд/боковик/высокая волатильность после новостей).
- **Спред:** используйте реалистичный для XAUUSD (20–35 пунктов). Скальпинг очень
  чувствителен к спреду и комиссии — обязательно включите комиссию вашего типа счёта.
- **На что смотреть в отчёте:**
  - **Profit Factor** > 1.3 (для скальпинга с учётом спреда/комиссии — важнее всего);
  - **Макс. просадка** — в пределах вашего `InpMaxDrawdownPct`;
  - **Средняя сделка** — должна уверенно превышать спред+комиссию, иначе стратегия
    «съедается» издержками;
  - **Sharpe Ratio** > 0.5 как ориентир;
  - **Кол-во сделок** — не менее **200–300** для статистической значимости;
  - равномерность кривой баланса (без единственной «сделки-героя»).
- **Про бюджет $15:** прогоните тест с начальным депозитом $15 и реальным плечом
  (например 1:500 у XM), чтобы увидеть реальные margin call и срабатывание
  `InpMinEquityStop`.

### Ограничения тестера (учтены в коде)
- **Новостной фильтр** (экономический календарь) в тестере не работает — код
  автоматически пропускает его (`MQL_TESTER`).
- **Telegram/Push** в тестере не отправляются (защита в `Notify`).
- Код **не использует блокирующих вызовов**, кроме `Sleep()` внутри retry
  (допустимо и в реале, и в тестере).

## 5. Чек-лист перед переходом на реальный счёт

- [ ] Компиляция без ошибок и предупреждений.
- [ ] Бэктест ≥ 1–2 года на «real ticks», Profit Factor и просадка приемлемы.
- [ ] **Форвард-тест на демо XM не менее 2–4 недель** с теми же настройками.
- [ ] Проверены дневной лимит убытка и полная остановка по просадке
      (можно временно занизить лимиты и убедиться, что бот встаёт на паузу).
- [ ] Проверен расчёт лота от риска на минимальном депозите (соответствие мин. лоту брокера).
- [ ] Проверена работа SL/TP, трейлинга, break-even и частичного закрытия на демо.
- [ ] Настроен и протестирован Telegram (пробное уведомление приходит).
- [ ] Проверено поведение на **XAUUSD** (крупный Point/спред) отдельно — при
      необходимости увеличены `InpMaxSpreadPips`, `InpAtrMaxPips`.
- [ ] Проверена реакция на requote/off quotes (retry-логика).
- [ ] Согласованы часы сессии с **временем сервера вашего брокера** (не локальным).
- [ ] Начинайте на реале с **минимального риска** (0.25–0.5%) и малого депозита.

## 6. Замечания по реализации

- **Точность цен:** все цены нормализуются через `NormalizePrice()` (Digits),
  лоты — через `NormalizeLot()` (volume step / min / max).
- **Обработка ошибок:** открытие и модификация повторяются при
  requote/price changed/timeout/connection; невосстановимые коды прекращают попытки.
- **Работа на закрытии бара:** сигналы вычисляются по закрытым барам (индекс 1),
  что исключает перерисовку и множественные входы внутри бара.
- **Метка частичного закрытия** хранится в глобальной переменной терминала
  `TMD_PC_<ticket>` и очищается при закрытии позиции.
- **Защита от повторной инициализации:** `ValidateInputs()` + один экземпляр на график;
  на разных графиках используйте разные Magic Number.
